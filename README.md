## Overview
Collection of tools, scripts, and analysis for **Digital Forensics & Incident Response (DFIR)**. 

This repository demonstrates practical skills in memory analysis, artifact parsing, election forensics, and cybersecurity investigations.

### Featured Case Study
**Digital Forensics & Cybersecurity Analysis of Oru West State Constituency 2019 Election**  
*(Forms EC8C & EC8E obtained via FOI request)*

**BSc Cyber Security Case Study**

Focusing on:
- Data integrity verification
- Chain of custody documentation
- Metadata analysis
- Electoral cybersecurity risks in Nigeria’s manual collation process

### Key Capabilities
- Memory forensics and dump analysis
- System artifact parsing and timeline generation
- Election data forensics and anomaly detection
- Custom Python scripts for DFIR workflows

### Technologies
- Python 3
- Volatility (or equivalent memory forensics tools)
- Pandas / NumPy for data analysis
- Git for version control
